DataManage Documentation


Welcome to Datamanage!

Starting the program

simply run the file called login (login.exe if you are using windows) and the following window will appear


DataManage Login window (above)

you then need to enter the login information required to connect to your database, if this information has been provided to you please enter it now.
if you do not have this information you need to get it from your database software.

Connecting to a database

Username and Password entries are straightforward, just your database username and password, the database name is the name of the database that you have either created or the one that has been created for you. the IP address is the location of your database on the network or the internet, if the database is running on your local machine then the IP address will be 127.0.0.1 or localhost. the port is the place where your database will send the information on a given IP address, on most machines running the postgresql database software (the recommended database software) the port will be 5432. once you have entered this information you can then press connect and DataManage will attempt a connection to your database.

if your login information is incorrect then the following error will appear


Login Error (above)

once you have successfully connected to a database you can then start entering information if you have a barcode reader simply scan an item, if the item already exists in the database then the program will show details about the item, if the item does not exist in the database then scanning the item will open the add item window and you will be asked to add the relevant information about the item Warning: you must fill all entries with information, if you do not have the information then simply enter a space into the box, you can then press save, the window closes and your database is refreshed to show the item. if you have more than one user connected (see MultiUser) then other users will have to press refresh to see the new items that have been added to the database.
