Welcome to Datamanage!

Starting the program

simply run the file called login (login.exe if you are using windows) and the following window will appear


DataManage Login window (above)

you then need to enter the login information required to connect to your database, if this information has been provided to you please enter it now.
if you do not have this information you need to get it from your database software.

Connecting to a database

Username and Password entries are straightforward, just your database username and password, the database name is the name of the database that
you have either created or the one that has been created for you. the IP address is the location of your database on the network or the internet, if the
database is running on your local machine then the IP address will be 127.0.0.1 or localhost. the port is the place where your database will send the
information on a given IP address, on most machines running the postgresql database software (the recommended database software) the port will
be 5432. once you have entered this infromation you can then press connect and DataManage will atempt a connection to your database.

if your login information is incorrect then the following error will appear


Login Error (above)
